Correct answer: Normal PR interval for this age

It's easy to apply adult PR-interval cutoffs to a pediatric ECG, but normal ranges shift meaningfully with age. In a 2-year-old, 90 ms sits comfortably within the normal reference range — it is neither a first-degree block (which requires prolongation beyond the age-adjusted upper limit) nor pre-excitation (which requires an interval shorter than the age-adjusted lower limit, typically paired with a delta wave). Always check interval norms against an age-specific reference table before flagging an abnormality.
